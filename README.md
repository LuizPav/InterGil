# This app is for manage the InterGil

It's a app made in React Native + Expo to manage internal championship.

here we use **Firebase** to manage the datas.

## How to run the project

### install _dependencies_

```sh
npm install

or

npm i
```

### Start Project

```tsx
 npx expo run:android --no-build-cache --device

 or

 npm run dev:build
```
